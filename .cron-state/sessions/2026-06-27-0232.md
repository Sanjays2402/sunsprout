## Notes / lessons

- **Scrolling-overlay section pattern (NEW):** for a non-blocking overlay that
  scrolls (achievements, quest-log), don't bolt sections onto the old row-index
  scroll. Flatten headers+rows into one DisplayItem[] and scroll over a FIXED
  BODY_H pixel budget so the panel height stays constant and a half-scrolled
  divider never leaves a gap. The shared-divider pattern now has 5 consumers.
- **One real bug, caught by a test:** the gift-glyph probe initially trusted any
  `dish-*` resolution, so `dish-frog` false-positived (the dish branch in
  bagGlyphForKey accepts compound premium keys and can't self-validate). Fixed
  by gating the dish probe on RECIPE_KEYS membership. tsc + the failing test
  caught it before the gate.
- **One real tsc diagnostic:** lovedGlyphKey missing from the RelationshipRow
  literal — caught by tsc, fixed in the same pass.
- **Repeated the #41 test-edit mistake ONCE:** a multi-line old_string that
  spanned an existing `it(...)` body clipped the `it('toggle open + close'` line
  in money-log.test.ts. Caught by re-read + corrective patch. CONFIRMED LESSON:
  anchor describe-block inserts on a UNIQUE single line (the bare
  `describe('X', () => {` header) and re-emit the whole block including the first
  it(), never a fragment that clips into the neighbour's body.
- **Stale LSP diagnostics again** (10th tick) — the patch tool echoed pre-edit
  errors at old line numbers; tsc exit 0 every time. Trust tsc.
