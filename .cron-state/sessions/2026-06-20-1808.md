## Design notes worth remembering
- Winter ordering: rain/sprinkler -> freeze -> advanceDay -> coop/dog/cat/greenhouse. Greenhouse re-asserts watered=true on inside crops AFTER advanceDay so they keep growing through winter.
- Pickaxe upgrade re-rolls the gem after `Pickaxe.strike()` returns; this keeps the state machine + existing mining tests untouched. Bias is multiplicative on catalog weights.
- Hangouts use Chebyshev radius 2 around per-NPC spots. 7-day cooldown per NPC after firing so the inbox doesn't refill instantly.
- Cat sits on `farmhouse.x + floor(w/2), farmhouse.y` and is rendered 8px above the tile centre so it visually perches on the roof.
