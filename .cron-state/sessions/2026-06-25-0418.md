## Features (one commit each)

1. **almanac-chip** (c7d5179) — `almanacHighlight()` / `highlightChipText()` pure
   helpers reuse `buildAlmanac`'s soonest-first ordering. New `ui/almanac-chip.ts`
   draws a one-line right-column pill under the sky dial (y=110) showing the single
   most-imminent event, but ONLY when it lands Today/Tomorrow. Kind-keyed colour
   rail + green "Today" text. Quiet stretches render nothing. 11 tests.

2. **minimap-pings** (56a7485) — `minimapPings(player, time)` derives pulsing rings
   from the SAME predicates the rest of the game uses: `canTurnIn` (board quest
   ready), `cartOpen` (Pip parked), `tournamentOpen && !alreadyEntered` (well).
   `MinimapPanel.draw` now threads player + time; renders expanding rings over the
   landmark pips + under the you-are-here dot, with a solid inner dot so the spot
   stays readable at the pulse trough. Empty array on a calm day. 7 tests.

3. **quality-heatmap** (22cade2) — `qualityHeat(streak, growthStages)` maps the
   streak->quality ladder onto five readable bands (dry / building / almost /
   silver / gold) with care-ramped alpha. New `ui/quality-heatmap.ts` washes every
   crop tile in WORLD space via a `(wx,wy)=>camera.worldToScreen` callback (same
   pattern as `drawBarks`) + a bottom-left legend, gated on `cropJournal.isVisible()`
   so it rides the `;` toggle — no new keybind (digit row is fully spent). 10 tests.

4. **sky-dial-weather** (7c908fe) — `skyWeatherStyle(weather)` pure deriver
   `{dimmed, cloud, storm}`. The widget dims + greys the sun/moon and tucks a small
   cloud puff (darker for storms) over it on rain/storm via `weatherToday()`, so the
   dial reads the sky, not just the clock. 4 new tests (19 in the sky-dial suite).

5. **hotbar-warning** (3c8becd) — `seedWarnLevel(count)` none/low/empty +
   `seedWarnPulse(level, nowMs)` deterministic sine breathe (empty pulses faster +
   floors brighter) in new pure `game/hotbar.ts`. `drawHUD`/`drawHotbar` gained an
   optional `nowMs` arg (defaults 0 so existing callers are unaffected); pulses an
   amber border on low/empty seed slots + tints the empty count badge amber, riding
   over the selection border without fighting it. 9 tests.
