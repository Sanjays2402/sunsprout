## Features (one commit each)
1. **toast-queue** (ade101c) — The engine held a SINGLE `toast` string + fade timer, so two messages fired the same frame clobbered all but the last writer (harvest + heart-up + achievement = you only saw one). New pure `ToastQueue` FIFO: each entry lives its own TTL, renderer draws up to 3 newest-first as a stack (freshest pill keeps the bright accent border, older ones dim + fade independently via `toastAlpha`), duplicate-text repeats refresh the newest entry's age instead of walling identical pills. Removed the now-dead `this.toast`/`this.toastFade` fields. 17 tests.
2. **help-overlay `?`** (9482b1d) — ~40 keybinds existed with ZERO in-game reference. `controls.ts` is the single source of truth, grouped Move/Farm/Gather/Animals/Upgrade/Home/Panels; `splitControlColumns` balances two columns; `HelpOverlay` renders a centered card with rounded key-caps. A test guards against emoji creeping in + against the new wayfinding keys (?,9,0) and panel toggles drifting out of sync. 13 tests.
3. **sky-dial** (eea66be) — HUD printed a clock but a number doesn't convey daylight-left. `sky-dial.ts` maps time onto sun-by-day / moon-by-night on a horizon-to-horizon arc (arcT 0..1 across the active window, altitude a sine hump peaking at the zenith, night arc wraps midnight monotonically). Widget draws the dotted arc + a pixel sun/crescent-moon + phase caption (Dawn/Midday/Dusk/Night) + an "Nh Nm left" readout, right-aligned below the weather strip. 15 tests.
4. **minimap `9`** (a44b2e6) — No whole-village view existed. `minimap.ts` projects the 40x30 world into a colour grid + derives labelled landmark markers from `world.buildings` + the cave. `MinimapPanel` blits the grid, glyph pips (Home/Shop/Inn/Well/Cave), a pulsing you-are-here dot, and a two-row legend. `projectTile` keeps tiles + markers pixel-aligned at any map size. 10 tests assert pond/tilled/building-centre placements against real worldgen.
5. **almanac `0`** (9ac8252) — "What's coming up" was scattered across festivals/birthdays/cart/tournament. `almanac.ts` aggregates every dated event within a 14-day horizon into one soonest-first list, reusing the 28-day calendar wrap. `AlmanacPanel` renders colour-railed rows with kind tag + title + detail + Today/Tomorrow/in-N-days countdown + Spring-7 date stamp + a calm empty state. 14 tests.
